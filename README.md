## ✨ 项目简介

这是一个基于纯前端技术栈开发的智能课程表系统，采用 2025 年最新的 UI 设计趋势，包含 **Glassmorphism（玻璃态）** 设计、流动渐变背景、3D 卡片悬浮效果等现代化视觉元素。系统无需后端支持，可直接部署在 GitHub Pages 上使用。

### 🎯 核心功能

- **📅 周次切换** - 支持选择不同教学周次查看课程安排
- **⭐ 今日高亮** - 自动识别并高亮显示当前星期的课程
- **📊 数据统计** - 实时统计今日课程数、本周课程数、任课教师数
- **🖨️ 打印功能** - 支持一键打印课程表，方便线下查看
- **📱 响应式设计** - 完美适配手机、平板、电脑等各种设备
- **🎨 玻璃态设计** - 采用 Glassmorphism 毛玻璃效果，视觉效果出众
- **⚡ 零依赖** - 纯原生 HTML/CSS/JavaScript，无需任何框架。
